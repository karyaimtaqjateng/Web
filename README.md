How to deploy (Netlify - drag & drop):
1. Go to https://app.netlify.com/drop and login (Google recommended).
2. Drag & drop the unzipped folder (the folder containing index.html).
3. Netlify will deploy and provide a URL like https://your-site.netlify.app.
4. To change the site name to karyaimtaq, go to Site settings > Change site name.

Notes & next steps:
- The project references font 'TeX Gyre Pagella'. If you want exact match, upload the font file into assets/fonts/ and update @font-face in style.css.
- Video is linked to Google Drive preview. If you prefer direct embed, replace the link with an iframe src using the Drive file ID: https://drive.google.com/file/d/FILE_ID/preview
- If any asset not showing, replace or add images to assets/ and redeploy.
